FLIGHT PLANNER CALCULATIONS

6.1 Using the plotter, measure and record the distances between regular checkpoints and use the Climb and Descent Data for distances to TOC and between TOD and BOD.

6.2 Calculate the remaining distance for each leg.

6.3 Mark the TOC, TOD, BOD on the Chart.

6.4 Fill out the TAS in the main Flight Planner section for each leg.

6.5 Fill out the Wind information in the main Flight Planner section for all checkpoints.

6.6 For each checkpoint, using True Course, TAS, Wind Information and your Flight Computer (back side), compute the Wind Correction Angle (WCA), Magnetic Heading (MH), and Ground Speed (GS).

6.7 With the Flight Computer, using Ground Speed (GS) and Leg Distance, compute the Estimated Time Enroute (ETE) for each leg. Add 10 minutes to the time for the BOD-Destination Airport leg.

6.8 With the Flight Computer, using Burn Rate (GPH from the Cruise Data) and Estimated Time Enroute (ETE), compute the Fuel Burnt for each leg.

6.9 Total up all the leg distances, ETEs, and fuel burnt and enter them at the bottom of the Flight Planner main section.
